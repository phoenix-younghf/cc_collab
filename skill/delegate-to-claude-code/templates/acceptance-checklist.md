# Acceptance Checklist

- [ ] Scope matches the delegated request and did not drift.
- [ ] Allowed write paths were respected.
- [ ] Required artifacts were generated and readable.
- [ ] Reported terminal state is explicit (`completed` or `inspection-required`).
- [ ] Declared verification commands were re-run by Codex.
- [ ] Verification output supports claimed completion.
- [ ] Closeout policy matches the observed result status.
- [ ] Follow-up actions are listed when state is `inspection-required`.
